Dear Students, In today’s lesson, we’ll go deeper into HTML (HyperText Markup Language). This is the foundation of every website. 📚 Theory 1. What is HTML? HTML gives structure to a webpage. Think of it as the skeleton of a website. It uses tags written inside < > to tell the browser what to display. 2. Common HTML Tags You Must Know → Tells the browser this is an HTML5 document. → The root of the page. → Info about the page (title, links to CSS, metadata). → Title shown in the browser tab. <body> → Everything visible on the page. <h1> … <h6> → Headings (from biggest to smallest). <p> → Paragraph. <a> → Link. <img> → Image. <ul>, <ol>, <li> → Lists (unordered/bulleted, ordered/numbered). <div> → A container/section. 🔧 Practical Step 1: Create a New Project Create a folder called class2-html.
